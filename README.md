# crypto-react

crypto-react is a basic crypto react app that get bids and ask, user can subscribe and unsubscribe to a particular currency.

# Demo
[demo app](https://crypto-react.netlify.com/)

# Prerequisite
* Node
* yarn or npm

# Usage
* yarn install
* yarn start

# Packages
* axios
* websocket
